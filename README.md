# 🎨 JavaScript Background Color Changer

A simple and beginner-friendly web project that allows users to change the background color of a webpage by entering a color name or hex code.

---

## 🚀 Features

* Change background color using:

  * Color names (e.g., `red`, `blue`)
  * Hex codes (e.g., `#ff5733`)
* Reset background color to default
* Clean and responsive UI
* Beginner-friendly JavaScript logic

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the webpage
* **CSS3** – Styling and layout
* **JavaScript (Vanilla JS)** – Functionality and interactivity

---

## 📂 Project Structure

```
project-folder/
│
├── index.html      # Main HTML file
├── style.css       # Styling file
└── script.js       # JavaScript logic
```

---

## ⚙️ How It Works

1. User enters a color name or hex code in the input field.
2. Clicking on the **color card** changes the background color.
3. Clicking the **Reset button** restores the default background color.

---

## 📸 Preview

* Input a color → Click on "Change the Background Color"
* Background updates instantly
* Click "Reset" → Original color restored

---

## 🧠 JavaScript Logic

* `getInput()` → Captures user input
* `changeColor()` → Applies background color
* `resetColor()` → Resets to default color
